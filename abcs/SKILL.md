# The ABCs of How We Learn

This skill turns the 26 research-backed learning mechanisms from *The ABCs of How We Learn* (Schwartz, Tsang & Blair, 2016) into a working toolkit. The user is a 9th–12th grade choir and guitar teacher; treat these letters as their shared professional vocabulary. Each letter is a **mechanism** (how learning happens), matched to **problems it solves** and **risks** that come with it.

> **Attribution.** The A–Z framework is inspired by Schwartz, Tsang & Blair's *The ABCs of How We Learn* (W. W. Norton, 2016). The reference files in `references/` and the summaries below are original explanations written for a high school music classroom, not reproductions of the book's text. Read the book for the full research and the authors' own treatment.

## How to work with this skill

1. **Diagnose before prescribing.** Identify the actual learning problem in the user's request (memory? perception? motivation? transfer? misconception?). The index below maps problems to letters.
2. **Read the reference file(s)** in `references/` for each letter you plan to apply — don't work from the one-line summary alone. Each file has: core mechanic, example, why it works, problems it solves, how to use it, and risks. Files are small; reading 2–4 is normal.
3. **Apply, don't recite.** Translate the mechanism into the user's context (choir rehearsal, guitar class, mixed-level high schoolers). Name the letter so the vocabulary sticks — e.g., "This is a Contrasting Cases (C) move: play the two recordings back to back."
4. **Always check the Risks section** of any letter you recommend and flag the relevant risk. The book treats every mechanism as having failure modes; so should you.
5. **Combine letters.** Real lessons stack mechanisms (e.g., J + C + F: struggle first, notice differences, then targeted feedback). Point out the combination explicitly.
6. **When designing whole lessons or units**, consult K (`references/k-knowledge.md`) for the sequencing spine: innovation/exploration experiences *first*, efficient solutions and drill *after* (see also J). K is an essay, not a recipe — read it when the question is about lesson architecture, assessment design, or "mastery vs. discovery" tradeoffs.

## Diagnostic index: problem → letter

| The problem sounds like... | Reach for |
|---|---|
| Can't grasp a new/abstract concept | **A** Analogy, **H** Hands On |
| Learned it here, can't use it there (no transfer) | **A** Analogy, **Q** Question Driven, **K** Knowledge |
| Confuses two similar things; can't hear/see the difference | **C** Contrasting Cases |
| Plateaued; repetition isn't producing improvement | **D** Deliberate Practice |
| Forgets material, names, vocabulary, sequences | **E** Elaboration, **G** Generation, **Z** ZZZs |
| Keeps making the same mistake; doesn't know how they're doing | **F** Feedback |
| Persistent wrong idea or faulty habit that resists correction | **U** Undoing |
| No idea where to start on a procedure | **W** Worked Examples, **O** Observation |
| Bored, checked out, "why do we have to learn this?" | **Q** Question Driven, **M** Making, **X** Excitement, **P** Participation |
| "I'm just not a music person" / gives up after failure | **Y** Yes I Can, **B** Belonging |
| Feels out of place; test anxiety; opts out of participating | **B** Belonging |
| Won't do the work at all; behavior/habit problem | **R** Reward |
| Reads/hears material but nothing sticks; shallow understanding | **S** Self-Explanation, **T** Teaching, **J** Just-in-Time Telling |
| Group work is dysfunctional or unequal | **L** Listening & Sharing, **N** Norms |
| Task is beyond their level but they want in | **P** Participation (scaffold & fade) |
| Overwhelmed by complex information; ideas are vague | **V** Visualization |
| Class culture/expectations are off | **N** Norms |
| Young or impulsive learners; over-scripted classroom | **I** Imaginative Play |
| Lecture lands flat; explanations feel abstract | **J** Just-in-Time Telling |

## The 26 letters at a glance

- **A — Analogy** (`references/a-analogy.md`): find the shared principle across surface-different examples; builds understanding and transfer.
- **B — Belonging** (`references/b-belonging.md`): feeling one belongs increases effort and quiets stereotype threat and alienation.
- **C — Contrasting Cases** (`references/c-contrasting-cases.md`): juxtapose near-misses so learners notice the distinguishing features. (Gold for ear training and technique.)
- **D — Deliberate Practice** (`references/d-deliberate-practice.md`): effortful, goal-directed practice just beyond current ability; the antidote to mindless reps.
- **E — Elaboration** (`references/e-elaboration.md`): connect new info to prior knowledge (acronyms, sentences, routes) so it can be found later.
- **F — Feedback** (`references/f-feedback.md`): make the gap between did and should-have-done visible and fixable.
- **G — Generation** (`references/g-generation.md`): retrieval practice — recall from partial cues, spaced over days, strengthens memory.
- **H — Hands On** (`references/h-hands-on.md`): perceptual-motor experience anchors abstract symbols and concepts.
- **I — Imaginative Play** (`references/i-imaginative-play.md`): pretend and role-play exercise symbolic thinking and cognitive control.
- **J — Just-in-Time Telling** (`references/j-just-in-time-telling.md`): let students wrestle with a problem *before* the explanation, so the telling lands.
- **K — Knowledge** (`references/k-knowledge.md`): the framing essay — efficiency vs. innovation, routine vs. adaptive expertise, and why innovation experiences should come first.
- **L — Listening and Sharing** (`references/l-listening-and-sharing.md`): structured collaboration; joint attention and group-worthy tasks.
- **M — Making** (`references/m-making.md`): producing an artifact or performance creates its own feedback-and-goals learning cycle and grows interest.
- **N — Norms** (`references/n-norms.md`): informal social rules shape what and how people learn; set them deliberately.
- **O — Observation** (`references/o-observation.md`): learning by watching and imitating, including affect; model failure and resilience too.
- **P — Participation** (`references/p-participation.md`): scaffold beginners into authentic activity, then fade the support.
- **Q — Question Driven** (`references/q-question-driven.md`): a driving question organizes curiosity, purpose, and well-connected memory.
- **R — Reward** (`references/r-reward.md`): shape behavior via reinforcement — with serious risks of killing intrinsic motivation; read the risks before using.
- **S — Self-Explanation** (`references/s-self-explanation.md`): prompt learners to explain material to themselves to build a mental model and expose gaps.
- **T — Teaching** (`references/t-teaching.md`): having students teach (peer tutoring, jigsaw) deepens the teacher-student's own understanding.
- **U — Undoing** (`references/u-undoing.md`): surface the misconception underneath the error and build a competing representation to replace it.
- **V — Visualization** (`references/v-visualization.md`): spatial representations (charts, maps, diagrams) reveal structure in complex information.
- **W — Worked Examples** (`references/w-worked-examples.md`): step-by-step demonstrations for initial procedural learning.
- **X — Excitement** (`references/x-excitement.md`): arousal focuses attention and stamps in memory; too much causes choking.
- **Y — Yes I Can** (`references/y-yes-i-can.md`): self-efficacy — praise effort not traits, use peer models, so learners engage and persist.
- **Z — ZZZs** (`references/z-zzzs-sleep.md`): sleep consolidates the day's learning; relevant to teenagers, performance prep, and cramming advice.

## Output expectations

- When building lesson plans, rehearsal plans, or materials, annotate which letters each segment uses and why (briefly — a parenthetical is enough).
- When troubleshooting a student problem, name the likely mechanism failure, cite the letter, and give a concrete classroom-ready move — not just theory.
- When the user asks about a letter directly, read that reference file and answer from it, translated into their music/arts context.
- Frequent power combinations for this teacher's context: **J→C→F** (invent/struggle, then contrast, then feedback) for concept lessons; **D+G+Z** for practice-routine design; **C+F** for ear training and technique diagnosis; **B+N+Y** for classroom culture; **M+P+T** for performance-based projects.
